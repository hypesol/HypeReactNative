Where is APK file in React Native
How to Generate a React Native Build APK for Android
react-native Create a shareable APK for android
Step-1 Create assets name folder in This dir(AwesomeProject\android\app\src\main)
       AwesomeProject is your project name
Step-2 Paste these line of code in Script section of package.json{

"prebuild": "npx react-native bundle --platform android --dev false --entry-file index.js --bundle-output android/app/src/main/assets/index.android.bundle --assets-dest android/app/src/main/res",
    "build": "cd android && gradlew assembleDebug && cd ../",
    "clear": "cd android && gradlew clean && cd ../",
}

step-3 Open Terminal and run these two cmd
{
1-:npm run clear
2-:npm run build
}
