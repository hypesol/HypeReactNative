  {id: '1', name: 'A' },
  {id: '2', name: 'B' },
]
MAP Function
Map function is used for data fetching or show data from array. But this function loads all data on first time which is not good practice.
{
DATA.map( item  => <Text key={item.id} style={{fontSize: 28}}> {item.name} </Text> )
}
FLATLIST
Best practice to fetch data from array or api.
<Flatlist 
	data = {DATA}
	renderItem = {(obj) => console.log(obj) }
	renderItem = {( {item} ) => console.log(item) }
renderItem = {( {item} ) => <Text> item.name </Text> }
/>
